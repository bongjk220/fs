팀협업중인 github에서 develop에 내가 작업한 feature/bjk를 머지 하고 싶어

GitHub에서 feature/bjk 브랜치를 develop에 머지하려면 보통 다음 과정을 거칩니다:

1. 최신 브랜치 가져오기
git checkout develop
git pull origin develop

2. 작업 브랜치로 이동
git checkout feature/bjk
git pull origin feature/bjk

3. 머지 실행
git checkout develop
git merge feature/bjk

4. 충돌 해결
만약 충돌(conflict)이 발생하면, 충돌 난 파일을 수정한 뒤:
git add <수정한_파일>
git commit

5. 원격 저장소에 푸시
git push origin develop
